# Smart Attendance Hub: Core Modules

## 專案定位
本專案為針對工廠端「承攬商管理」需求所建置的 C# Proof of Concept。
目前專案專注於後端檢核邏輯的獨立驗證，將身分比對流程模組化，以此作為後續建置完整智慧出勤系統的基礎架構。

## 核心功能實作
* **承攬商身分驗證模組** (`ContractorValidation.cs`)：
  * 透過物件導向設計 `Contractor` 資料模型。
  * 模擬接收影像辨識端 (如 YOLO) 的推論結果 (ID)。
  * 實作與資料庫名單的比對邏輯，並依據授權狀態，啟動對應的放行或警報機制。

## 下階段技術整合計畫 (Next Steps)
1. **架構升級與容錯設計**：從 Mock Data 測試轉向實體SQL資料庫串接，並導入 Exception Handling 以確保系統的強健性。
2. **跨語言串接**：研究 Python (影像推論端) 與 C# (管理邏輯端) 的資料交換機制，如：API串接。
3. **客製化前端介面開發**：規劃將核心邏輯整合至GUI介面，以符合承攬商管理的實際操作需求。
