# Smart Attendance Hub: Core Modules

## 專案定位
本專案為針對工廠端「承攬商管理」需求所建置的 C# 核心邏輯驗證區 (Proof of Concept)。
在完成 C# 基礎語法與物件導向 (OOP) 概念的先期研讀後，我著手將理論轉化為實際的模組化程式碼，作為後續開發完整智慧出勤系統的底層基礎。

## 核心功能實作
* **承攬商身分驗證模組** (`ContractorValidation.cs`)：
  * 透過物件導向設計 `Contractor` 資料模型。
  * 模擬接收影像辨識端 (如 YOLO) 的推論結果 (ID)。
  * 實作與後端名冊的快速比對邏輯，並依據授權狀態給予對應的通行與警報判定。

## 下階段技術整合計畫 (Next Steps)
1. **架構升級**：為應對大規模廠區的資料吞吐量，計畫將現有的線性查詢結構優化。
2. **跨語言串接**：研究 Python (影像推論端) 與 C# (管理邏輯端) 的高效通訊機制。
3. **GUI 介面開發**：待核心模組穩固後，將邏輯整合至 Windows 視窗應用程式中，實現客製化前端介面。
